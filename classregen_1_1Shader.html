Detailed Description

a piece of code that is executed on the GPU.

Encapsulates a GLSL program, helps compiling and linking together the shader stages.

void Shader::enableAttributes ( RenderState *  rs)

Enables attributes attached to shader. Note: this is not called in enable because VAO's should be used to enable attributes. If you enable instanced attributes with this function you might have to manually set the attribute divisor to 0 again. Also note that you have to bind manually the index buffer when using this function for indexed geometry (normally VAO binds index buffer too).
