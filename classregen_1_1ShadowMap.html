Detailed Description

Implements shadow mapping for directional,spot and point Light's.

Parallel-Split-Shadow-Mapping (PSSM) is used for directional lights. Omnidirectional shadow mapping is done using a depth cubemap.

See Also
http://http.developer.nvidia.com/GPUGems3/gpugems3_ch10.html
