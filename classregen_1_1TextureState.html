enum regen::TextureState::MapTo

Defines what is affected by the texture.

Enumerator
MAP_TO_COLOR 

The texture is combined with the fragment color.

MAP_TO_DIFFUSE 

The texture is combined with the result of the diffuse lighting equation.

MAP_TO_AMBIENT 

The texture is combined with the result of the ambient lighting equation.

MAP_TO_SPECULAR 

The texture is combined with the result of the specular lighting equation.

MAP_TO_SHININESS 

The texture defines the glossiness of the material.

MAP_TO_EMISSION 

The texture is added to the result of the lighting calculation.

MAP_TO_LIGHT 

Lightmap texture (aka Ambient Occlusion).

MAP_TO_ALPHA 

The texture defines per-pixel opacity.

MAP_TO_NORMAL 

The texture is a normal map..

MAP_TO_HEIGHT 

The texture is a height map..

MAP_TO_DISPLACEMENT 

Displacement texture. The exact purpose and format is application-dependent..

MAP_TO_CUSTOM 

user defined.
