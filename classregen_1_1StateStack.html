Detailed Description

template<typename StackType, typename ValueType, typename ApplyFunc>
class regen::StateStack< StackType, ValueType, ApplyFunc >

A stack that keeps track of a state value.

Redundant state switches are avoided. ValueType must implement default constructor ValueType() and the != operator. The stack can be locked. Pushes do not change the state value until the stack is locked.
