Detailed Description

template<class T>
class regen::ref_ptr< T >

Adds auto reference Management to a pointer.

Note that only ref_ptr should manage the memory of the data pointer. Else you get double free corruption. Do not instantiate two ref_ptr with the same data pointer. Copying a ref_ptr is valid.
