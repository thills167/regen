GLenum regen::StencilFunc::func_

specifies the test function. Accepted values are GL_NEVER,GL_LESS,GL_LEQUAL,GL_GREATER,GL_GEQUAL,GL_EQUAL, GL_NOTEQUAL,GL_ALWAYS. The initial value is GL_ALWAYS.

GLuint regen::StencilFunc::mask_

specifies a mask that is ANDed with both the reference value and the stored stencil value when the test is done. The initial value is all 1's.

GLint regen::StencilFunc::ref_

specifies the reference value for the stencil test. The initial value is 0.
