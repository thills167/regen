string regen::NodeAnimation::Channel::nodeName_

The name of the node affected by this animation. The node must exist and it must be unique.

Position key frames.

Behavior regen::NodeAnimation::Channel::postState

Defines how the animation behaves after the last key was processed. The default value is ANIM_BEHAVIOR_DEFAULT (the original transformation matrix of the affected node is taken).

Behavior regen::NodeAnimation::Channel::preState

Defines how the animation behaves before the first key is encountered. The default value is ANIM_BEHAVIOR_DEFAULT (the original transformation matrix of the affected node is used).
