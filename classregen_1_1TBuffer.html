Detailed Description

Transparency-Buffer state.

Transparency requires direct shading and most techniques require z sorted geometry.

Member Enumeration Documentation

enum regen::TBuffer::Mode

Transparency mode.

Enumerator
MODE_FRONT_TO_BACK 

Front to back blending. Geometry should be sorted back to front.

MODE_BACK_TO_FRONT 

Back to front blending. Geometry should be sorted front to back.

MODE_SUM 

Use add blending. No sorting required.

MODE_AVERAGE_SUM 

Use average add blending. Count samples per texel and divide result by sample count in post pass.
