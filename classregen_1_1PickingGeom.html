Detailed Description

Implements geometry picking.

Uses special geometry shader that checks if the mouse intersects the processed face. If an intersection occurred the object id, instance id and face depth are written to output variables. These outputs are captured using transform feedback. Finally a CPU depth test is used to obtain the front-most face intersection.
